# Patrones de Diseño en PHP – Ejercicios (Factory, Adapter, Decorator, Strategy)

## Estructura
- **Factory:** Crea personajes según dificultad (Esqueleto en fácil, Zombi en difícil).
- **Adapter:** Win10 acepta archivos legados de Win7 gracias a un adaptador.
- **Decorator:** Añade armas a personajes y modifica su daño/ataque.
- **Strategy:** Muestra un mismo mensaje en consola, JSON y archivo TXT.

## Requisitos
- PHP 8+

## Cómo ejecutar las demos
```bash
php src/Factory/demo.php
php src/Adapter/demo.php
php src/Decorator/demo.php
php src/Strategy/demo.php
```
